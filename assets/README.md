# PyHerdr brand assets

*herd your terminals · multi-agent multiplexer*

## Palette
- Matte black `#0E1318`
- Python blue `#306998 → #4B8BBE`
- Python yellow `#FFD43B → #FFE873`
- Slate (terminal surfaces) `#222B36`
- Font: **JetBrains Mono** (Fira Code / Cascadia Code / Consolas fallback)

The mark: a blue **agent** pane (the eye) + a dark pane + a yellow fold + a `>_` prompt — a mini bot orchestrating terminals.
